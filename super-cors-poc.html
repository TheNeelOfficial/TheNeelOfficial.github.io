Technical Details

Vulnerable Configuration:

Access-Control-Allow-Origin: *
Access-Control-Allow-Credentials: true

W3C Specification Violation:

According to the W3C Fetch Specification:

"If the credentials flag is true and the Access-Control-Allow-Origin header value is the wildcard *, return fail."
